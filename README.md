CHE Hacks
==========================
CHE Hacks is a hackathon at the [Cherry Hill Public Library](https://goo.gl/SsJoqy).

Requirements
============
*	[Node.js](https://nodejs.org/download/)
*	[Compass](http://compass-style.org/install/)

```
$ gem update --system
$ gem install compass
```

Running
=======
```
$ cd app
$ npm install
$ node app.js
```
